# 用户指南

欢迎使用 TinyIoTHub 用户指南。本章节将帮助你了解系统的各项功能。

## 目录

- [设备管理](./devices) - 设备的添加、配置和管理
- [驱动管理](./drivers) - 驱动程序的安装和配置
- [告警管理](./alarms) - 告警规则和通知设置
- [用户管理](./users) - 用户账号和权限管理

## 核心概念

### 设备 (Device)

设备是 TinyIoTHub 的核心实体，代表物理或虚拟的 IoT 设备。每个设备具有：

- **唯一标识**: SN 序列号
- **驱动类型**: 支持的协议（Modbus、ONVIF 等）
- **配置参数**: 连接参数和数据点
- **状态**: 在线/离线/告警

### 驱动 (Driver)

驱动是设备与系统通信的桥梁。TinyIoTHub 支持：

- Modbus RTU/TCP
- ONVIF (视频监控)
- SNMP
- Ping
- 自定义驱动

### 告警 (Alarm)

告警系统实时监控设备数据，支持：

- 阈值告警
- 状态变化告警
- 离线告警
- 多通道通知（MQTT、邮件等）

### 模板 (Template)

设备模板简化设备创建过程，预定义：

- 驱动类型
- 配置参数
- 数据点映射
- 告警规则
