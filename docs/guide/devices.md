# 设备管理

设备管理是 TinyIoTHub 的核心功能，用于管理所有接入的物联网设备。

## 添加设备

### 方式一：手动添加

1. 登录管理后台
2. 进入「设备管理」页面
3. 点击「添加设备」按钮
4. 填写设备信息：
   - 设备名称
   - 序列号 (SN)
   - 选择驱动类型
   - 配置连接参数
5. 点击「保存」

### 方式二：使用模板

1. 进入「设备模板」页面
2. 选择合适的模板
3. 点击「使用模板创建设备」
4. 根据向导填写配置

## 设备状态

| 状态 | 说明 |
|------|------|
| 🟢 在线 | 设备正常连接 |
| 🔴 离线 | 设备断开连接 |
| 🟡 告警 | 设备触发告警 |
| ⚪ 未知 | 状态未知 |

## 设备操作

### 查看详情

点击设备列表中的设备，可查看：
- 基本信息
- 实时数据
- 历史记录
- 告警记录

### 编辑设备

支持在线编辑设备配置，修改后自动生效。

### 删除设备

> ⚠️ 删除设备会同时删除相关数据，请谨慎操作！

### 远程控制

根据驱动支持情况，可对设备进行：
- 参数读取
- 命令下发
- 固件更新

## 数据采集

设备支持多种数据采集模式：

- **主动上报**: 设备主动推送数据
- **定时轮询**: 系统定时读取
- **事件触发**: 状态变化时采集
