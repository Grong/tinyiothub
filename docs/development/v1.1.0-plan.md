## 数据库设计

### device_data 表 (新创建)
```sql
CREATE TABLE device_data (
    id TEXT PRIMARY KEY,
    device_id TEXT NOT NULL,
    property_name TEXT NOT NULL,
    property_value TEXT NOT NULL,
    property_type TEXT NOT NULL, -- number, string, boolean
    unit TEXT,
    quality TEXT DEFAULT 'good', -- good, bad, uncertain
    timestamp TEXT NOT NULL,
    created_at TEXT NOT NULL DEFAULT (datetime('now')),
    FOREIGN KEY (device_id) REFERENCES devices(id) ON DELETE CASCADE
);

CREATE INDEX idx_device_data_device_time ON device_data(device_id, timestamp DESC);
CREATE INDEX idx_device_data_property ON device_data(device_id, property_name);
```

## API 设计

### 历史数据 API
- `GET /api/v1/devices/{id}/data` - 获取设备历史数据
- `GET /api/v1/devices/{id}/data/latest` - 获取最新数据
- `DELETE /api/v1/devices/{id}/data` - 删除历史数据

### 邮件发送 API
- `POST /api/v1/notifications/channels/{id}/send` - 发送通知
- `POST /api/v1/notifications/test` - 发送测试
